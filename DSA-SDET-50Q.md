## PATTERN IDENTIFICATION GUIDE (For 8+ Years)

### When to Use Sliding Window?
- Substring/subarray problems
- Contiguous elements
- Optimization with two pointers
- **Red flags:** "shortest", "longest", "minimum", "maximum" subarray/substring

### When to Use Two Pointers?
- Sorted array problems
- Two elements meeting condition
- In-place manipulation
- **Red flags:** "pair sum", "palindrome", "merge", "partition"

### When to Use BFS/DFS?
- Tree/graph traversal
- Level-order or path problems
- Connected components
- **Red flags:** "traversal", "path", "connected", "island"

### When to Use Heaps?
- K-th largest/smallest
- Top K elements
- Merging K sorted structures
- **Red flags:** "Kth", "top K", "merge K", "median"

### When to Use DP?
- Overlapping subproblems
- Optimal substructure
- State-based decisions
- **Red flags:** "max/min", "count ways", "can we reach", "longest/shortest"

### When to Use Backtracking?
- Permutations/combinations
- Board problems (Sudoku, N-Queens)
- Exhaustive search with constraints
- **Red flags:** "all permutations", "combinations", "board puzzle", "valid solutions"

### When to Use Greedy?
- Optimization without revisiting
- Exchange argument possible
- Local best = global best
- **Red flags:** "interval", "scheduling", "jumps", "activity selection"
